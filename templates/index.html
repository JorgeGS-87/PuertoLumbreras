      🌐 Servicios OGC disponibles
      WMS — Web Map Service 1.3.0
      Visualiza las capas como imágenes georreferenciadas. Compatible con QGIS, ArcGIS y cualquier cliente OGC.
      GET /wms?SERVICE=WMS&REQUEST=GetCapabilities
      GetCapabilities GetMap GetFeatureInfo
      WFS — Web Feature Service 2.0.0
      Descarga features vectoriales en GeoJSON o GML 3.2. Soporta filtros por BBOX y límite de features.
      GET /wfs?SERVICE=WFS&REQUEST=GetCapabilities
      GetCapabilities DescribeFeatureType GetFeature
      🗂️ Capas expuestas
      Nombre Tipo Descripción
      georuta:vias LineString Red viaria OSM
      georuta:puntos Point Puntos de interés
      georuta:portales Point Numeración postal
      🛠️ Conexión desde QGIS
      WMS: Capa → Añadir capa WMS/WMTS
      http://localhost:5000/wms
      WFS: Capa → Añadir capa WFS/OGC API
      http://localhost:5000/wfs
      📋 Ejemplos de consulta
      Obtener vías en GeoJSON:
      /wfs?SERVICE=WFS&REQUEST=GetFeature&TYPENAME=georuta:vias&outputFormat=application/json&COUNT=100
      Filtrar por BBOX:
      /wfs?SERVICE=WFS&REQUEST=GetFeature&TYPENAME=georuta:puntos&BBOX=-1.88,37.53,-1.75,37.60&outputFormat=application/json
